# Setup

## Prerequisites

- Python 3

## Configuration

Set the `UNIFI_API_KEY` env var, or create `config.json` next to SKILL.md (gitignored). Start from `config.json.example`.

| Variable | Required | Description |
|---|---|---|
| `UNIFI_API_KEY` | Yes | UniFi Site Manager API key |
| `UNIFI_BASE_URL` | No | API base URL (default: `https://api.ui.com`) |
